# 사용법

우선 자산 탭에서 본인의 현재 소유 자산들을 정리한 뒤에, 자산 도우미 채팅에 "자산 분석" 등의 키워드를 입력하면 본인 자산을 전체적으로 분석해서 진단해줘요. 그리고, "배분 조언" 등의 키워드를 입력하면 "현금 비율이 너무 많아요. 투자 비율을 조금 더 늘려보세요." 라는 형식으로, 본인이 돈을 어떻게 관리하면 좋을지에 대한 어드바이스를 출력해요. 아 참, "빚 점검" 키워드를 입력하면 빚의 총합, 자산에 대한 빚의 비율, 진짜 내가 가지고 있는 돈 (대출은 자산에서 마이너스로 계산돼요.) 을 정리해서 보내줘요. ✔️

수입/지출 탭에서 급여나 고정 용돈 + 고정 지출, 월정액 등을 정리한 뒤 자산 도우미 채팅에 "남는 돈" 과 같은 키워드를 입력하여 본인의 남는 돈을 계산하여 매달 얼마를 모아야 1년에 얼마가 모이는지 계산해줘요. 저축에 활용해보세요! 📊

목표 탭에서 본인이 모으고 싶은 저축액, 그 기간과 현재까지 모은 돈의 정보를 작성한 뒤에 자산 도우미 채팅에 "목표 달성" 과 비슷한 키워드를 전송하면 목표에 대해서 기간을 조금 늘려보라고 하는 등의 조언을 해줍니다. 효과적인 저축을 해보세요! 🥇

아 참, 민감한 주제에 대해선 답변하기 어려워요 :) 🈲

청소년이나 노약자 등을 위한 큰 글 모드도 있어요!

AI 모듈이나 API를 활용하지 않고 자체 로직으로 이루어져 있습니다!
